Bangla Font came with Avro Keyboard

To keep the package size small and updated, this version of Avro Keyboard came with two UNICODE compliant open type Bangla font - Siyam Rupali and Kalpurush, and an ANSI based Bangla font - Kalpurush ANSI

You can download other Bangla fonts separately from OmicronLab's online reserve. Download them from here: http://www.omicronlab.com/go.php?id=4 )

 

Installing downloaded Bangla fonts:

On Windows 2000: Copy>Paste downloaded fonts to: C:\WinNT\Fonts\ folder

On Windows Xp, 2003, Vista, 7: Copy>Paste downloaded fonts to: C:\Windows\Fonts\ folder
